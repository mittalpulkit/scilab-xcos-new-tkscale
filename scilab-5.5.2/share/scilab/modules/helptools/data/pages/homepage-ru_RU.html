Что нового в Scilab 5.5.1?

Перестроен программный код возможностей создания графического интерфейса:

  • Улучшения функций создания графического окна с новыми свойствами.
  • Новые стили элементов графического интерфейса: tab, layer, spinner.
  • новые свойства элементов графического интерфейса.
  • Можно управлять размещением и изменением размеров элементов графического интерфейса с помощью макетов.

Этот релиз также включает в себя несколько наборов функций:

  • Модуль HDF5, позволяющий чтение/запись файлов HDF5 и массивов данных.
  • Java-интерфейс внешних объектов для взаимодействия с Java-объектами напрямую из Scilab'а. Более подробную информацию смотрите в кратком руководстве.
  • Функция getURL для получения/чтения удалённых файлов через протоколы ftp:// или http://.
  • Возможность переводить внешние модули Scilab.
  • Возможности MPI, доступные в дереве исходников Scilab (не активированы по умолчанию).
  • Новые специальные функции erfi и dawson.
  • Новая программа решения daskr в Scilab и Xcos.

Внедрены новые графические возможности, улучшающие удобство пользования и возможности Scilab:

  • Улучшения возможностей функции Matplot для отображения изображений в истинных цветах.
  • Управление освещением на трёхмерных графиках.
  • Новые взаимодействия с графическими окнами, позволяющие править дескрипторы графических окон с помощью мыши прямо в графических окнах.
  • Возможности ярлыков данных переписаны на Java.
  • Возможности настройки осей и меток с соответствующими свойствами.

Scilab перешёл на версию 2.1 лицензии CeCILL.

Также в этом релизе исправлено множество ошибок и улучшена документация.
